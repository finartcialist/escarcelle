about

The word escarcelle is the name of the purse that used to be worn at the belt.

Escarcelle is an invitation to barter a personal object with high sentimental value, in exchange for a similar object but belonging to a stranger.

how it works?

Here are the steps to barter an object on the curated market:

  • gain an idea of the current state of the market on the Escarcelle website (upcoming).
  • if there is an object you like, you will need to find one to offer in exchange. select an object with high sentimental value that belongs to you.
  • now is the time to gain some bartering credit by filling up the submission form. all souvenirs are not made equal. in order to keep a certain quality assurance of objects on the market, we need to make sure that every one of them is relevant.
  • if your object was selected, you will receive an email giving you all the needed information to access the market.
  • access the market and select the object you wish you receive in exchange for yours. we will put it aside.
  • send your object by mail.
  • after receiving your object, we will send you the one you selected by mail.

beta version

In order to make sure that Escarcelle is ready for a future public presentation, we would like to test the exchange protocol and to make sure that every software component of the project works as we want.

With this in mind, there will be a first run of the beta version of the project during Fall 2021.

If you wish to participate, please fill this form to be chosen for the beta version. We will pick a few participants amongst the answers received in order to start the first cohort.
